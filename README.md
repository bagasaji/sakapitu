# InteriorWeb - Website Desain Interior

Website untuk layanan desain interior dan bangunan.

## Fitur
- Halaman portfolio desain
- Paket layanan (Standard, Premium, Bestseller)
- Kontak via WhatsApp
- Server Node.js dengan Express

## Teknologi
- HTML, CSS, JavaScript
- Node.js + Express
- Git + GitHub

## Instalasi
```bash
npm install
npm start
